# my-linux-notes

### To print current working directry

```bash
pwd
```

### To create new folder

```bash
mkdir <folder name>
```
### To open folder

```bash
cd <foldername>
```
### To see list of items in the folder

```bash
ls
```

### To create new file

```bash
touch <filename.txt>
```

### To open a text file

```bash
vi <filename>
```

### To copy a .txt file

```bash
cp <filename.txt> <nameofnewfile.txt>
```

### To rename a file

```bash
mv <filename> <newname>
```

### To move a file

```bash
mv <filename> <foldername>
